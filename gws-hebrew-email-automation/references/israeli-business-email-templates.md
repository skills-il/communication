# Israeli Business Email Templates for Freelancers

## Quote / Proposal (הצעת מחיר)

### Subject: הצעת מחיר - [שם הפרויקט]

```
שלום [שם הלקוח],

תודה על הפגישה/השיחה שלנו. מצורפת הצעת מחיר עבור [תיאור הפרויקט].

פרטי ההצעה:
- היקף: [תיאור מפורט]
- לוח זמנים: [X] ימי עבודה
- עלות: [סכום] ש"ח (לא כולל מע"מ 18%)
- תנאי תשלום: שוטף + 30

ההצעה בתוקף עד [תאריך].

אשמח לכל שאלה.

בברכה,
[שם]
[טלפון]
```

## Invoice Follow-up (מעקב חשבונית)

### Subject: בנוגע לחשבונית [מספר]

```
שלום [שם],

רציתי לוודא שחשבונית מס' [מספר] מיום [תאריך] התקבלה בהצלחה.

סכום: [סכום] ש"ח
תנאי תשלום: שוטף + [30/45/60]
תאריך פירעון: [תאריך]

אם יש צורך בפרטים נוספים, אשמח לסייע.

תודה,
[שם]
```

## Project Update (עדכון פרויקט)

### Subject: עדכון שבועי - [שם הפרויקט]

```
שלום [שם],

להלן עדכון לגבי התקדמות [שם הפרויקט]:

מה הושלם השבוע:
- [משימה 1]
- [משימה 2]

בתכנון לשבוע הבא:
- [משימה 3]
- [משימה 4]

נקודות לתשומת לב:
- [נקודה]

אשמח לשמוע הערותיך.

בברכה,
[שם]
```

## Meeting Request (בקשת פגישה)

### Subject: בקשה לפגישה - [נושא]

```
שלום [שם],

אשמח לקבוע פגישה לדון ב[נושא].

זמנים פנויים מצדי:
- יום [ראשון/שני/שלישי/רביעי/חמישי], [תאריך] בשעה [שעה]
- יום [ראשון/שני/שלישי/רביעי/חמישי], [תאריך] בשעה [שעה]

הפגישה צפויה לארוך כ-[X] דקות.

מה מתאים לך?

תודה,
[שם]
```

## Payment Confirmation Request (בקשת אישור תשלום)

### Subject: אישור קבלת תשלום - חשבונית [מספר]

```
שלום [שם],

אודה אם תוכל/י לאשר שהתשלום עבור חשבונית [מספר] בוצע.

פרטים:
- סכום: [סכום] ש"ח
- תאריך שליחה: [תאריך]

תודה רבה,
[שם]
```

## Notes on Hebrew Email Etiquette

- **Greeting**: "שלום" (formal), "היי" (informal), "שלום רב" (very formal)
- **Closing**: "בברכה" (regards), "תודה" (thanks), "בהצלחה" (good luck)
- **Days**: ראשון (Sun), שני (Mon), שלישי (Tue), רביעי (Wed), חמישי (Thu), שישי (Fri), שבת (Sat)
- **Business hours**: 09:00-18:00 Sun-Thu, 09:00-13:00 Fri
- **Avoid sending**: Friday after 14:00, all day Saturday, Jewish holidays
- **Currency**: Always write amounts with ש"ח after the number
- **VAT**: Always specify כולל מע"מ (including) or לא כולל מע"מ (excluding)
- **Date format**: DD.MM.YYYY (Israeli standard)
