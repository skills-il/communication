# Hebrew Communication Templates

Ready-to-use message templates for common Israeli business situations. Copy, fill in the brackets, and send.

All templates follow Israeli communication norms: direct, warm where appropriate, formal only when required.

---

## Invoice and Payment

### Invoice Reminder - Day 1 (Friendly)

**Use when:** Due date just passed, good relationship with client.

```
היי [שם],

רציתי להזכיר שחשבונית מספר [X] על סך ₪[סכום] הגיעה לפירעון ב-[תאריך].

אשמח שתסדיר בנוח.

תודה,
[שמך]
```

---

### Invoice Reminder - Day 14 (Polite Follow-up)

**Use when:** No response to first reminder, still want to maintain the relationship.

```
שלום [שם],

מזכיר שוב בנוגע לחשבונית מספר [X] על סך ₪[סכום] מתאריך [תאריך], שלא שולמה עדיין.

נשמח לקבל את התשלום בהקדם. אם יש בעיה מצדכם, אשמח לדבר.

[שמך]
[טלפון]
```

---

### Invoice Reminder - Day 30 (Firm)

**Use when:** Multiple follow-ups ignored, need to be clear without burning bridges.

```
שלום [שם],

חשבונית מספר [X] על סך ₪[סכום] עדיין לא שולמה, למרות התזכורות הקודמות מתאריכי [תאריך 1] ו-[תאריך 2].

אבקש לסדר את התשלום עד [תאריך] לכל המאוחר.

לפרטים נוספים: [מספר טלפון].

[שמך]
```

---

### Invoice Reminder - Final Notice (Pre-legal)

**Use when:** Payment is significantly overdue and you are considering legal action. Only use after the firm reminder has also gone unanswered.

```
לכבוד [שם החברה / שם איש הקשר],

למרות פניות חוזרות, חשבונית מספר [X] על סך ₪[סכום] מתאריך [תאריך] לא שולמה.

זוהי הודעה אחרונה לפני שאפנה לייעוץ משפטי לגבי גביית החוב.

לתשלום מיידי או לבירורים, ניתן לפנות אלי עד [תאריך]:
טלפון: [מספר]
מייל: [כתובת]

[שמך]
[שם עסקך]
```

## Client Communication

### Project Kickoff Confirmation

**Use when:** Confirming project start details with a new client.

```
היי [שם],

שמחתי לדבר! מסכם בקצרה מה סיכמנו:

פרויקט: [שם הפרויקט]
היקף: [תיאור קצר]
לו"ז: [תאריך התחלה] עד [תאריך סיום]
תמורה: ₪[סכום] [מועד תשלום: מקדמה/מסירה/חודשי]
תשלום ראשון: [תאריך ואמצעי תשלום]

אם הכל תואם, אשלח הסכם/הצעת מחיר רשמית.

[שמך]
```

---

### Deadline Extension Request (to client)

**Use when:** You need more time and want to be upfront about it.

```
היי [שם],

רציתי לעדכן אותך לפני שמגיע הדדליין:

[תיאור קצר של הסיבה] גרם לעיכוב קל.

אני מעריך שאסיים עד [תאריך חדש]. ניתן לקבל את זה, או שיש לך אילוץ שלא ידעתי עליו?

[שמך]
```

---

### Proposal Follow-up

**Use when:** You sent a proposal and haven't heard back in 5-7 business days.

```
היי [שם],

שלחתי לך את ההצעה ב-[תאריך]. רציתי לבדוק אם הגיעה ואם יש שאלות.

אם הכיוון לא מתאים, אשמח גם לדעת - עוזר לי להבין.

[שמך]
```

---

### Upsell or Additional Work

**Use when:** You want to suggest more work to an existing client without being pushy.

```
היי [שם],

במהלך העבודה שלחנו שמתי לב ש[הזדמנות או בעיה שראית].

זה לא חלק ממה שסיכמנו, אבל חשבתי שכדאי לציין. אם מעניין לדבר על זה, אשמח.

[שמך]
```

## Government and Official Bodies

### Opening a File or Request to a Government Office

**Use when:** Writing a formal request to any Israeli government body (Tax Authority, Bituach Leumi, municipality, etc.).

```
לכבוד [שם הגוף/המחלקה],

שם מלא: [שמך]
ת.ז. / ח.פ.: [מספר]
טלפון: [מספר]
מייל: [כתובת]

הנדון: [נושא הפנייה בקצרה]

[תיאור הבקשה: 2-4 משפטים ברורים. מה אתם מבקשים, ומה הרקע הרלוונטי. לא צריך לכתוב ספר - עובדות ובקשה.]

מצורפים: [רשימת מסמכים, אם יש]

בתודה,
[שמך המלא]
[חתימה אם נדרש]
תאריך: [DD/MM/YYYY]
```

---

### Appeal to Tax Authority (רשות המיסים)

**Use when:** Appealing an assessment, requesting a payment extension, or disputing a fine.

```
לכבוד פקיד השומה / רשות המיסים,

שם: [שמך]
ת.ז. / ח.פ.: [מספר]
תיק מס: [מספר אם ידוע]

הנדון: ערר / בקשה בנוגע ל[נושא]

ברצוני [לערור על / לבקש / להבהיר] את הנושא הבא:

[תיאור העניין - עובדות בלבד, בסדר כרונולוגי]

[הסבר מדוע הבקשה מוצדקת]

מצורפים: [מסמכים]

בכבוד,
[שמך המלא]
תאריך: [DD/MM/YYYY]
```

---

### Request to Bituach Leumi (ביטוח לאומי)

**Use when:** Updating status, requesting information, or disputing a determination.

```
לכבוד המוסד לביטוח לאומי,

שם: [שמך המלא]
מספר זהות: [ת.ז.]
מספר ביטוח לאומי: [אם ידוע]
טלפון ליצירת קשר: [מספר]

הנדון: [נושא]

[פירוט הבקשה בצורה ברורה וממוקדת. כלול תאריכים רלוונטיים ומספרי תיקים אם יש.]

אנא עדכנו אותי לגבי [מה שאתם מצפים לקבל - אישור, החלטה, מסמך וכו'].

בתודה,
[שמך]
תאריך: [DD/MM/YYYY]
```

## Supplier and Vendor

### Requesting a Quote

```
שלום [שם],

אני [שמך] מ[שם עסקך].

רציתי לבקש הצעת מחיר עבור [תיאור מה שצריך, כמה שיותר ספציפי: כמות, מפרט, מועד].

מועד הדרוש: [תאריך]
תקציב משוער (אם יש): [סכום]

אשמח לקבל הצעה עד [תאריך].

תודה,
[שמך]
[טלפון]
```

---

### Cancellation or Delay Notification to Supplier

```
שלום [שם],

רציתי לעדכן שנאלצנו [לדחות / לבטל] את [ההזמנה / הפרויקט / הפגישה] שנקבעה ל-[תאריך].

הסיבה: [תיאור קצר וכן].

[אם דחייה: "הצעתי לתאם מחדש ל-[תאריך אלטרנטיבי] אם מתאים לכם."]
[אם ביטול: "אעביר אישור ביטול רשמי בנפרד אם נדרש."]

מתנצל על אי הנוחות.

[שמך]
```

## Internal and Team

### Slack/WhatsApp Update on Task Status

**Use when:** Updating a team or manager on where things stand.

```
עדכון ל[שם הפרויקט/משימה]:

מצב: [בתהליך / הושלם / תקוע]
מה נעשה: [1-2 שורות]
מה הבא: [השלב הבאה]
צפי: [תאריך / "בהמשך השבוע"]

[אם יש בעיה: "יש חסימה: [מה הבעיה וממי צריך עזרה]"]
```

### Meeting Request (Internal)

```
היי [שם],

רציתי לקבוע [שיחה / פגישה] קצרה בנושא [נושא].

[אם מציעים זמן:] יש לך [יום X בשעה Y] או [יום X בשעה Y]?

לא צריך יותר מ-[X דקות].

[שמך]
```

## Tips for Using These Templates

1. **Always fill in all brackets.** Sending a message with [שם] still in it looks unprofessional.
2. **Adjust tone to match your relationship.** If you know someone well, "שלום" can become "היי" and the message can be shorter.
3. **Keep it short.** Israelis prefer direct communication. If your message is more than 5-6 lines, consider whether all of it is necessary.
4. **Date format.** Always DD/MM/YYYY or written out (3 באפריל 2026).
5. **Currency.** ₪ before the number: ₪1,234.56.
6. **For government letters:** Formal templates above are starting points. Complex matters (appeals, disputes) should be reviewed by an accountant or lawyer.
